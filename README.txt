Simple Banking System
Skapat av Fredrik Sjöbäck, Daniel Svendsen och Jimmy Emanuelsson.

Bygg ditt egna simple banking system steg för steg:

(OBS) För att bygga och köra programmet behöver du ha installerat Git och Maven på din dator.

1, Öppna valfri terminal och navigera dig till den katalog du vill bygga ditt projekt.

2, Skriv in 'git clone https://github.com/jimmsi/Grupp5SimpleBankingSystem' i terminalen.

3, 'cd Grupp5SimpleBankingSystem'

4, 'mvn compile'

5, 'mvn test'

6, 'mvn package'

7, 'java -cp target/Grupp5SimpleBankingSystem-1.0-SNAPSHOT.jar se.yrgo.Main' för att köra programmet.
